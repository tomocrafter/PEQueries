PEQueries
====

## Description
これは、簡単にPocketMine-MPのQueryを誰でも扱えるようにするライブラリです。  
商用利用可能ですが、ご使用いただく際はサイトなどに  
私の名前やこのライブラリの名前を掲載させていただければ光栄です。  
06/07 テスト完了しました！

## Usage
使い方は、このライブラリに同梱されている  
example.php を読んでみてください！
